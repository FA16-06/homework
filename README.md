# FA16-06 Homework

This repository will hold group homework assignments.

# Making PDFs

To build all of the PDFs:

```console
$ make depend
$ make
```

If you also want to view the PDFs in your browser, you can run:

```console
$ make server
```

Or to build and run the server:

```console
$ make run
```
